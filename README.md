# Login Attempt Control System

## Description
A Python-based login system that controls repeated incorrect password attempts.  
If the user enters the wrong password multiple times, the system temporarily locks access to prevent unauthorized login attempts.

## Features
- Checks user password
- Counts failed login attempts
- Allows limited login attempts
- Temporarily locks the account after maximum failures
- Automatically unlocks after a waiting period
- Prevents brute-force attacks

## Technology Used
- Python

## Concepts Used
- Loops
- Conditional statements
- Variables
- Time module
- User input handling
